# AC Signal Parameters

Alternating Current (AC) signals periodically reverse direction. The fundamental waveform is the Sine Wave.

## Key Parameters

### Period ($T$)
The time it takes to complete one full cycle.
*   **Unit**: Seconds ($s$).
*   **Formula**: $T = \frac{1}{f}$

### Frequency ($f$)
The number of cycles per second.
*   **Unit**: Hertz ($Hz$).
*   **Formula**: $f = \frac{1}{T}$

### Angular Frequency ($\omega$)
Rotation speed in radians per second.
*   **Formula**: $\omega = 2\pi f$

### Amplitude
*   **Peak Value ($U_{max}$)**: The maximum deviation from zero.
*   **Peak-to-Peak ($U_{pp}$)**: Difference between positive and negative peaks ($2 \times U_{max}$).

### Phase ($\varphi$)
The position of a wave within its cycle ($0^\circ$ to $360^\circ$).
*   **Phase Difference**: The shift between two waves.
    *   **In Phase**: Peaks align ($\Delta \varphi = 0^\circ$).
    *   **Anti-phase**: Peak aligns with Trough ($\Delta \varphi = 180^\circ$).
    *   **Quadrature**: $90^\circ$ shift.
