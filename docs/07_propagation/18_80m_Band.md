# 80m Band (3.5 MHz)

A workhorse band for regional communications.

## Characteristics
*   **Frequency:** 3.5 - 4.0 MHz.
*   **[Propagation](14_Solar_Cycle_&_Propagation.md):**
    *   **Day:** D-layer absorption limits range to regional (200-300 miles).
    *   **Night:** Global DX possible (500-1000+ miles).
*   **[Noise](../01_electricity/26_AC_Signals_&_Noise.md):** High (S6-S9). Thunderstorm static in summer.
*   **[NVIS](11_Near_Vertical_Incidence_Skywave_NVIS.md):** Excellent for [Near Vertical Incidence Skywave](11_Near_Vertical_Incidence_Skywave_NVIS.md) (regional coverage without skip zone).

## Usage
*   **Best For:** Evening ragchewing, Regional Nets, Emergency Comms.
*   **Antennas:** Large (~125 ft dipole).
*   **Activity:** Very high population.
