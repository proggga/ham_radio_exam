# AC Signal Parameters

Alternating Current (AC) signals periodically reverse direction. The fundamental waveform is the Sine Wave.

## Key Parameters

### Period ($T$)
The time it takes to complete one full cycle.
*   **Unit**: Seconds ($s$).
*   **Formula**: $T = \frac{1}{f}$

### Frequency ($f$)
The number of cycles per second.
*   **Unit**: Hertz ($Hz$).
*   **Formula**: $f = \frac{1}{T}$

### Angular Frequency ($\omega$)
Rotation speed in radians per second.
*   **Formula**: $\omega = 2\pi f$

### Amplitude
*   **Peak Value ($U_{max}$)**: The maximum deviation from zero. **Exam Definition:** The "Amplitude" of a sine wave is strictly defined as the **Peak Value** ($U_{max}$), not Peak-to-Peak or RMS.
*   **Peak-to-Peak ($U_{pp}$)**: Difference between positive and negative peaks ($2 \times U_{max}$).
*   **Average Value ($U_{avg}$)**: Average over one half-cycle (for sine).
    *   $U_{avg} = \frac{2}{\pi} U_{max} \approx 0.637 U_{max}$.
    *   *Exam Tip:* The average value of a current over a quarter period ($0$ to $\pi/2$) or half period ($0$ to $\pi$) is $\frac{2}{\pi} \times I_{max}$.
    *   *Note:* Moving-coil meters measure the Average value but the scale is calibrated to show RMS (assuming a sine wave). If measuring a non-sine wave (e.g. square wave), the reading will be incorrect.

### Phase ($\varphi$)
The position of a wave within its cycle ($0^\circ$ to $360^\circ$).
*   **Phase Difference**: The shift between two waves.
    *   **In Phase**: Peaks align ($\Delta \varphi = 0^\circ$).
    *   **Anti-phase**: Peak aligns with Trough ($\Delta \varphi = 180^\circ$).
    *   **Quadrature**: $90^\circ$ shift.
