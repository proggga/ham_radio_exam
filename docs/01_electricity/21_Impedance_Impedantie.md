# Impedance (Impedantie)

Impedance ($Z$) is the total opposition to current flow in an AC circuit, combining Resistance ($R$) and [Reactance](../02_components/07_Reactance_Reactantie.md) ($X$).
*   **Unit**: Ohm ($\Omega$).

## Vector Addition
Because Resistance and [Reactance](../02_components/07_Reactance_Reactantie.md) are $90^\circ$ out of phase, they must be added as vectors (Pythagoras).

### Series Circuits
$$Z = \sqrt{R^2 + X^2}$$

*   *[Exam](../12_regulations/02_Licensing_in_the_Netherlands.md) Tip (Pythagoras Triples):* Look out for the **3-4-5** ratio (and multiples like 6-8-10, 24-32-40, 30-40-50).
    *   If $R=3$ and $X=4$, then $Z=5$.
    *   If $R=300$ and $X=400$, then $Z=500$.
    *   This saves calculation time during the exam.

### Parallel Circuits
$$\frac{1}{Z} = \sqrt{(\frac{1}{R})^2 + (\frac{1}{X})^2}$$
