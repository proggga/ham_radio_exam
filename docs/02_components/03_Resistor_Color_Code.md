# Resistor Color Code

Most axial resistors use colored bands to indicate their resistance value and tolerance.

## The 4-Band Code
1.  **Ring 1**: 1st Digit
2.  **Ring 2**: 2nd Digit
3.  **Ring 3**: Multiplier (Number of Zeros)
4.  **Ring 4**: Tolerance

| Color  | Digit | Multiplier | Tolerance |
| :---   | :---: | :---       | :---      |
| Black  | 0     | $1$        |           |
| Brown  | 1     | $10$       | 1%        |
| Red    | 2     | $100$      | 2%        |
| Orange | 3     | $1k$       |           |
| Yellow | 4     | $10k$      |           |
| Green  | 5     | $100k$     | 0.5%      |
| Blue   | 6     | $1M$       |           |
| Violet | 7     | $10M$      |           |
| Grey   | 8     |            |           |
| White  | 9     |            |           |
| Gold   |       | $0.1$      | 5%        |
| Silver |       | $0.01$     | 10%       |

## Example
**Yellow, Violet, Orange, Gold**
*   Yellow (4), Violet (7), Orange (3 zeros).
*   $47,000 \Omega = 47 k\Omega$.
*   Gold = $\pm 5\%$.

## Mnemonic (Dutch)
**Z**ij **B**rengt **R**ozen **O**p **G**errits **G**raf **B**ij **V**ies **G**rijs **W**eer.
(Zwart, Bruin, Rood, Oranje, Geel, Groen, Blauw, Violet, Grijs, Wit).
