# SWR Meter (Staandegolfmeter)

An [SWR](../06_antennas/16_Standing_Wave_Ratio_SWR.md) meter measures the **Standing Wave Ratio** on a transmission line, indicating how well the antenna system is matched to the transmitter.

## Operation
*   **Placement**: Connected between the Transmitter and the [Antenna](../10_safety/03_Antenna_&_Tower_Safety.md) (or Antenna Tuner).
*   **Method**: Directional couplers sample the **Forward Power** ($P_{fwd}$) and **Reflected Power** ($P_{ref}$).

## Usage Procedure
1.  Select **FWD** (Forward).
2.  Key the Transmitter (using a constant carrier mode like [FM](../01_electricity/35_Frequency_Modulation_FM.md) or [CW](../01_electricity/33_CW_Abbreviations_&_Prosigns.md)).
3.  Adjust the calibration knob to the **SET** mark (Full scale).
4.  Select **REF** (Reflected).
5.  Read the SWR value on the scale.

## Interpretation
*   **1:1**: Perfect match (No reflection).
*   **1.5:1**: Good match.
*   **> 2:1**: High reflection. Usually requires tuning or indicates a fault. High SWR can damage the transmitter PA.
